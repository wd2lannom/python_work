# GIT Commands

git pull - get current code

```bash
git pull 
```

git status - current status

```bash
git status
```

git add . - Add any files to version control

```bash
git add . 
```

git commit -a -m '' - commit code
```bash
git commit -a -m 'COMMENT HERE'
```

git push - push to GitHub

```bash
git push
```
